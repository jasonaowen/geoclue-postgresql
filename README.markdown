# geoclue-postgresql

A simple service to listen for geoclue events and log them to PostgreSQL.

## Building

This project uses the [CMake](https://cmake.org/) build system.

```ShellSession
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./geoclue-psql
```
